浮动

视觉格式化模型，大体上将页面中盒子的排列分为三种

  1. 常规流
  2. 浮动
  3. 定位

应用场景

  1. 文字环绕
  2. 横向排布

浮动的基本特点

修改 float 属性值为：

left:左浮动，元素靠上靠左

right:右浮动，元素靠上靠右

默认值为 none

  1. 当一个元素浮动后，元素必定为块盒（display 更改为 block）
  2. 浮动元素的包含块，和常规流一样，为父元素的内容盒

盒子尺寸

  1. 宽度为 auto 时，适应内容宽度
  2. 高度为 auto 时，适应内容高度
  3. margin 为 auto，为 0
  4. 边框、内边距、百分比设置与常规流一样
💡

除了高度的百分比之外，其他的百分比都是相对于包含快的宽度
高度的百分比是相对于包含块的高度

盒子排序

  1. 左浮动盒子靠上靠左

  2. 右浮动盒子靠上靠右

  3. 浮动盒子在包含快中排列时，会避开常规流块盒

    <div class="container">
  <div class="normal"></div>
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>

  4. 常规流块盒在排列时，无视浮动盒子

    <div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="normal"></div>
</div>

  5. 行盒在排列时，会避开浮动盒子

    但是下面是不是违反了这项规定

    <div class="container">
  <img src="" alt="" style="float:left" />
  <p>lorem100</p>
</div>

    注意：如果文字没有在行盒中，浏览器会直接生成一个行盒包裹文字，叫做匿名行盒

解决浮动高度坍塌

  1. 使用 clearfix 技术

    .clearfix::after {
  content: "";
  display: table;
  clear: both;
}

  2. 使用 overflow 属性

    .parent {
  overflow: hidden/auto;
}
