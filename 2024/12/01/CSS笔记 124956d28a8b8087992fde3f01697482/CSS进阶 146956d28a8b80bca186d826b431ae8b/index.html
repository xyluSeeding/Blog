web 字体

以下是字体前置知识

p {
  font-family: aaa bbb ccc; // 字体类型
}
// 先看有没有aaa,再看有没有bbb，最后看有没有ccc

解决电脑上没有安装相应字体，强制用户安装字体

使用 @font-face 制作一个新字体

/*制作一个新的字体，名称叫做good night */
@font-face {
  font-family: "good night";
  src: url("./font/晚安体.ttf");
}
p {
  font-family: "good night";
}
