2024/12/1 html

列表元素

有序列表

ol：ordered list

li：list item

// type: 前面排序取值 1(阿拉伯),a(小写),A(大写),i(小写罗马),I(大写罗马) //
除非前面逻辑上很重要，一律不用 type ,用 css 属性写 // reversed 反序
<ol type="1" reversed>
  <li>开门</li>
  <li>进去</li>
  <li>关门</li>
</ol>
效果: 3. 关门 2. 进去 1. 开门

无序列表

ul：unordered list

li：list item

<ul>
  <li>开门</li>
  <li>进去</li>
  <li>关门</li>
</ul>
效果: ● 开门 ● 进去 ● 关门

定义列表

通常用于一些术语的定义

dl：deinition list 类 ul

dt：deinition title 标题

dd：deinition description 描述

<dl>
  <dt>HTML1</dt>
  <dd>123</dd>

  <dt>HTML2</dt>
  <dd>456</dd>

  <dt>HTML3</dt>
  <dd>789</dd>
</dl>
