span[无语义]

没语义，仅用于设置样式

三原色包含：
<span style="color: red">红</span>
<span style="color: green">绿</span>
<span style="color: blue">蓝</span>

以前：某些元素在显示时会独占一行（块级元素），而某些元素不会换行（行级元素)
到了 HTML5，已经弃用这种说法。

pre

预格式化文本元素

空白折叠：在源代码中的连续空白字符 （空格、换行、制表），会被折叠成一个空格

例外：在 pre 元素中的内容不会出现空白折叠

在 pre 元素内部出现的内容，会按照代码格式显示到页面上

该元素通常用于在网页中显示一些代码

pre 元素功能的本质：它有一个默认的 css 样式（white-space:pre;）

<p>Lorem, ipsum dolor.</p>
=>Lorem, ipsum dolor.

<pre>
Lorem,        
                    ipsum dolor.</pre
>
=>Lorem, ipsum dolor.

通常在页面上显示代码时会在pre套一个code ,code 表示代码区域

<code>
  <pre>
            var i = 0;
            i++;
        </pre
  >
</code>
