html笔记

2024/12/1 html

a 元素

a 元素：超链接 ⇒ 可以从一个文档跳转到另一个文档

href 属性

  1. 跳转地址(普通链接)
<a href="https://www.baidu.com">百度</a>
  1. 跳转到某个锚点(锚链接)

((h2[id="chapter$"]>{章节$})+p>lorem)*6 a[href="#chapter$"]*6 ...
<a href="#chapterX">章节x</a>
...
<h2 id="chapterX">章节X</h2>
<p>xxxxxx</p>
...

id 属性：全局属性，表示元素在文档中的唯一编号

  1. 跳转其他页面的锚点(锚链接)
<a href="2.跳转到某个锚点.html#chapterX">章节x</a>
  1. 点击后触发某个功能(功能链接)

— 执行 JS 代码

<a href="javascript:alert('你好')">弹出你好</a>

— 发送邮件

<a href="mailto:xxxxxxxxx@qq.com">点击给我发送邮件</a>
<!-- 要求用户计算机安装有邮件发送软件:exchange  -->

— 拨打电话

<a href="tel:xxxxxxxxx)">拨打电话</a>
<!-- 要求用户计算机安装拨号软件，或移动端点击  -->

target 属性

表示跳转窗口位置

target 取值：

_self：在当前页面窗口打开

_blank：在新窗口打开

<a href="https://www.baidu.com" target="_self">百度</a>
